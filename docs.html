Retrieve paginated tasks

Fetch tasks belonging to the authenticated user, supporting optional search, status filtering, and pagination.

Authorizations:
BearerAuth
query Parameters
status
string
Enum: "Pending" "In Progress" "Completed"

Filter tasks by status

search
string

Search tasks by title (substring)

page
integer
Default: 1

Page number

limit
integer
Default: 10

Page size limit

Responses

Development Server

http://localhost:5000/api/tasks

Response samples

  • 200
  • 401
Content type
application/json
{
  • "success": true,
  • "message": "Tasks retrieved successfully",
  • "data": {
    • "tasks": [
      • {
        • "id": 1,
        • "userId": 1,
        • "title": "Design Database Schema",

        }
      ],
    • "totalTasks": 15,
    • "totalPages": 2,
    • "currentPage": 1,
    • "limit": 10
    }
